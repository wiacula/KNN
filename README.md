KNN Classifier: Custom vs Scikit-learn Implementation
This project compares a custom implementation of the K-Nearest Neighbors (KNN) algorithm with the scikit-learn library's KNN classifier on the MNIST dataset (handwritten digits).

Features:
Custom KNN implementation in Python.
Comparison with scikit-learn's KNN.
Visualization of classification results and confusion matrices.
Unit tests for the custom KNN implementation.
Usage:
Train and test the classifiers on the MNIST dataset.
Analyze performance using classification reports and visualizations.
Requirements:
Python 3.x
NumPy, scikit-learn, matplotlib
